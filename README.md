프로젝트 소개
백엔드는 **Java(Spring Boot)**와 JPA를 사용하여 상품(재고) 관리, 주문 처리, 회원 인증(JWT) 등 쇼핑몰의 핵심 비즈니스 로직을 견고하게 구현한 RESTful API 서버로 구축했습니다.

프론트엔드는 Vue.js와 Vue Router를 활용하여, 페이지 이동 없이 동적으로 콘텐츠가 변경되는 SPA(Single Page Application)로 구축했습니다. 

이를 통해 사용자가 상품을 검색하고, 장바구니에 담고, 결제하는 과정을 매끄럽게 경험할 수 있도록 설계했습니다.
